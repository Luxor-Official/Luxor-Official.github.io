Setup

Learn how to install and use Luxor assets inside your Roblox experience. This guide walks you through installation, integration, and best practices.

Installation

Pick one of the below methods to install Luxor assets:

Manual

You can download your purchased Luxor assets directly through Updatr. Import the model files into ServerScriptService or any preferred directory.

Advanced

Luxor assets can also be synced using Git submodules and Rojo. This method is recommended for teams using version control.

Warning

Warning

Please do not modify the source code of Luxor assets. Use the provided API and configuration options instead. Modifying source code makes it harder to receive updates.

Basic Usage

Once installed, you can load your Luxor asset using the Loader module:

local Loader = require(game.ServerScriptService.Luxor.Loader) Loader:Init({ LicenseKey = "YOUR_KEY_HERE", Product = "ModeratorCall" })
