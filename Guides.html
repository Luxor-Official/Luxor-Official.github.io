Getting Started

Installation

Pick one of the methods below to install your Luxor product.

Using Updatr

Download the latest version from your Updatr dashboard after purchasing the asset. Place the package inside your game and follow the configuration guide.

Recommended Structure
ServerScriptService └── Luxor ├── Config ├── Modules └── Main

Configuration

Every Luxor product includes a configuration module that lets you customize permissions, logging, UI, and other product specific settings.

Option Description
Permissions Who can access commands.
Logging Discord webhook support.
UI Customize the appearance.
Modules Enable optional features.
Important

Do not modify the core source code of Luxor products. Instead, use the documented configuration options to ensure future updates work correctly.

Updating

Whenever a new version is released, simply download the latest build from Updatr and replace the old package while keeping your configuration.
